# AUTOHOTKEY PIXEL DETECTION SCRIPT (EDUCATIONAL)

## DESCRIPTION

THIS AUTOHOTKEY SCRIPT DEMONSTRATES PIXEL COLOR DETECTION AND AUTOMATED MOUSE INPUT AROUND THE CENTER OF THE SCREEN. IT INCLUDES TOGGLE, HOLD, AND OFF MODES, A SMALL GUI STATUS INDICATOR, SOUND FEEDBACK, AND AN OPTIONAL FAST-CLICK FEATURE.

THE PROJECT IS DESIGNED TO SHOW HOW INPUT OPTIMIZATION, PIXELSEARCH, HOTKEYS, TIMERS, AND GUI ELEMENTS WORK TOGETHER IN AUTOHOTKEY.

---

## FEATURES

* PIXEL COLOR SCANNING NEAR SCREEN CENTER
* TOGGLE / HOLD / OFF MODES
* CONFIGURABLE PIXEL BOX SIZE AND SENSITIVITY
* MINIMAL ALWAYS-ON-TOP GUI
* SOUND FEEDBACK FOR MODE CHANGES
* OPTIONAL FAST-CLICK MODE
* OPTIMIZED FOR SPEED AND LOW LATENCY
